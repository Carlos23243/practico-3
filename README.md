# PracticaEstructuraDatos - Gestión de Biblioteca
Este proyecto es una aplicación de consola en C# diseñada para demostrar el uso eficiente de estructuras de datos no lineales para la gestión de inventarios. El enfoque principal es la optimización de búsquedas y el manejo de registros únicos mediante diccionarios y conjuntos.
# Características
# Registro de Libros: 
Almacenamiento basado en el estándar ISBN.

# Validación de Duplicados: 
Evita la duplicidad de registros utilizando claves únicas.

# Gestión de Categorías: 
Clasificación automática de géneros literarios sin repeticiones.

# Análisis de Rendimiento: 
Medición del tiempo de ejecución mediante la clase
